# Aplicación de Cálculo de Factorial — Android

## Descripción

Esta aplicación móvil fue desarrollada en Android Studio utilizando Kotlin, con el objetivo de calcular el factorial de un número ingresado por el usuario. El proyecto forma parte de una práctica académica orientada al aprendizaje de estructuras de control, manejo de datos numéricos y desarrollo de aplicaciones móviles.

El sistema permite capturar un número entero, procesarlo mediante lógica de programación y mostrar el resultado correspondiente al cálculo del factorial dentro de la interfaz de usuario.

---

## Funcionalidad principal

La aplicación solicita al usuario un número entero y, al ejecutar la acción de cálculo, obtiene el factorial correspondiente. El resultado se muestra en pantalla de forma clara, permitiendo al usuario visualizar el valor resultante del proceso.

El sistema también considera la validación de datos, evitando entradas inválidas y asegurando que el cálculo se realice únicamente con valores adecuados.

La aplicación presenta las siguientes características principales:

- Interfaz gráfica simple e intuitiva.
- Entrada de datos mediante campo de texto.
- Validación de entrada numérica.
- Cálculo del factorial de un número entero.
- Presentación del resultado en pantalla.
- Ejecución en dispositivos Android o emulador.

---

## Lógica implementada

El cálculo del factorial se basa en la multiplicación sucesiva de números enteros desde 1 hasta el valor ingresado por el usuario. Dependiendo de la implementación, el cálculo puede realizarse mediante un proceso iterativo o recursivo.

Asimismo, se considera el manejo de casos especiales, como el factorial de 0, cuyo resultado es 1.

El sistema también contempla limitaciones relacionadas con el tamaño de los números, debido a que valores grandes pueden provocar desbordamiento (overflow) en los tipos de datos utilizados.

El objetivo del proyecto fue desarrollar una aplicación móvil capaz de realizar cálculos matemáticos básicos, reforzando el uso de estructuras de control, validación de datos y desarrollo de interfaces en Android.
